# Teleport SDK

## Running the tests

Fund an account with:

- Rinkeby ETH
- Rinkeby Arbitrum ETH
- Kovan ETH
- Kovan Optimism ETH
- Goerli ETH
- Goerli Optimism ETH
- Goerli Arbitrum ETH

then add `USER_PRIV_KEY=your-account-private-key` to your `.env`. You can then run the tests using:

```sh
yarn test
```
